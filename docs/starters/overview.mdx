## Getting Started

<Steps>
  <Step title="Choose Your Template">
    Pick Flask for Python or Next.js for TypeScript
  </Step>
  
  <Step title="Clone & Install">
    ```bash
    # Flask
    cd flask-minimal
    pip install -r requirements.txt
    
    # Next.js
    cd nextjs-minimal
    pnpm install
    ```
  </Step>
  
  <Step title="Configure Environment">
    Set up your API token and scenario ID
  </Step>
  
  <Step title="Run Locally">
    ```bash
    # Flask
    python app.py
    
    # Next.js
    pnpm dev
    ```
  </Step>
  
  <Step title="Deploy">
    Deploy to Vercel or your preferred platform
  </Step>
</Steps>
