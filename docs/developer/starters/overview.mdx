## Getting Credentials

Both templates require ToughTongue AI credentials:

<Steps>
  <Step title="Create Account">
    Sign up at [ToughTongue AI](https://www.toughtongueai.com/)
  </Step>
  
  <Step title="Get API Key">
    Visit [Developer Portal](https://app.toughtongueai.com/developer?tab=api-keys)

  </Step>
  
  <Step title="Get Scenario ID">
