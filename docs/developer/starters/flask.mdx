A lightweight Python-based starter template using Flask for backend and Preact for frontend. Perfect for developers who prefer Python and want a simple, no-build-tools approach.

## Overview

The Flask starter provides:
- **Python + Flask** backend with API proxy routes
- **Preact** frontend (lightweight React alternative)
- **No build tools** required - pure HTML/CSS/JS
- **Vercel-ready** serverless deployment

---

## Quick Start (Preview)

<Steps>
  <Step title="Install Dependencies">
    ```bash
    cd flask-minimal
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
  </Step>
  
  <Step title="Configure Environment">
    ```bash
    cp .env.sample .env
    ```
    
    Add your ToughTongue AI API key:
    ```env
    TTAI_TOKEN=your_api_key_here
    ```
  </Step>
  
  <Step title="Run the App">
    ```bash
    python app.py
    ```
    
    Visit [http://localhost:8008](http://localhost:8008)
  </Step>
</Steps>
