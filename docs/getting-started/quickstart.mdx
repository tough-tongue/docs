### Create Your First Agent

<Steps>
  <Step title="Sign Up">
    Create a free account at [app.toughtongueai.com](https://app.toughtongueai.com)
  </Step>
  <Step title="Open Creator">
    Go to [Scenario Studio](https://app.toughtongueai.com/advanced-scenario-create)
  </Step>
  <Step title="Describe">
    Example: "Create a sales coach for objection handling. The prospect should be skeptical about
    pricing."
  </Step>
  <Step title="Test & Share">Test your scenario, refine it, share the link.</Step>
</Steps>

## For Developers

### Get API Key

1. Create account at [app.toughtongueai.com](https://app.toughtongueai.com)
2. Go to [Developer Portal → API Keys](https://app.toughtongueai.com/developer?tab=api-keys)
3. Create and copy your token

### Integration Options

<Tabs>
  <Tab title="Iframe">
```html
<iframe
  src="https://app.toughtongueai.com/embed/YOUR_SCENARIO_ID?bg=black"
  width="100%"
  height="600px"
  allow="microphone"
></iframe>
```

Params: `bg`, `name`, `color` — [Full guide →](/guides/embedding)

  </Tab>

  <Tab title="REST API">
```bash
# Create scenario
curl -X POST https://api.toughtongueai.com/api/public/scenarios \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Cold Call", "prompt": "You are a VP..."}'
```

[API reference →](/api-reference/overview)

  </Tab>

  <Tab title="Starter Templates">
```bash
# Flask
cd flask-minimal && pip install -r requirements.txt && python app.py

# Next.js

cd nextjs-minimal && pnpm install && pnpm dev

````

[Templates guide →](/starters/overview)
  </Tab>
</Tabs>

### Handle Events

```javascript
window.addEventListener("message", (event) => {
  if (event.data.event === "onStop") {
    console.log("Session ended:", event.data.sessionId);
  }
});
````
