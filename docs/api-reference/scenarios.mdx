# Scenarios API

Manage conversation scenarios programmatically.

## List Scenarios

`GET /scenarios`

Retrieve all scenarios you have access to.

```javascript
const response = await fetch(
  'https://api.toughtongueai.com/api/public/scenarios',
  {
    headers: { 'Authorization': 'Bearer YOUR_API_TOKEN' }
  }
);

const scenarios = await response.json();
```

## Create Scenario

`POST /scenarios`

Create a new scenario.

**Request Body:**

```json
{
  "name": "Customer Support Training",
  "description": "Practice handling difficult customers",
  "ai_instructions": "You are a frustrated customer...",
  "is_public": true,
  "is_recording": true
}
```

**Response:**

```json
{
  "id": "scenario_abc123",
  "name": "Customer Support Training",
  "created_at": "2024-01-15T10:00:00Z",
  ...
}
```

## Get Scenario

`GET /scenarios/{scenario_id}`

Retrieve a specific scenario.

## Update Scenario

`PATCH /scenarios/{scenario_id}`

Update scenario properties.

## Delete Scenario

`DELETE /scenarios/{scenario_id}`

Permanently delete a scenario.

<Warning>
  Deleting a scenario also deletes all associated sessions. This cannot be undone.
</Warning>
