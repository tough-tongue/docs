---
title: "Create Access Token"
api: "POST /scenario-access-token"
description: "Create a short-lived access token for embedding scenarios"
---

Create a Scenario Access Token (SAT) for embedding private scenarios in iframes.
Tokens are valid for 1-24 hours.

## Request Parameters

<ParamField body="scenario_id" type="string" required>
  The scenario ID to create an access token for
</ParamField>

<ParamField body="duration_hours" type="integer" default="1">
  Token validity duration (1-24 hours)
</ParamField>

<ParamField body="email" type="string">
  Optional user email. For org context, must be format `*@ORG-ID.toughtongueai.com`
</ParamField>

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `access_token` | string | JWT token for authenticating iframe access |
| `expires_at` | string | ISO 8601 timestamp when the token expires |
| `iframe_src` | string | Pre-built iframe URL with the token included |

<RequestExample>
```json
{
  "scenario_id": "693960de75f58a7b74c835bf",
  "duration_hours": 4
}
```
</RequestExample>

<ResponseExample>
```json 200
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_at": "2026-01-05T13:37:09.233722Z",
  "iframe_src": "https://app.toughtongueai.com/embed/693960de75f58a7b74c835bf?scenarioAccessToken=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```
</ResponseExample>

## Usage

The `iframe_src` field contains a ready-to-use URL that you can set directly as the
`src` attribute of your iframe:

```html
<iframe 
  src="https://app.toughtongueai.com/embed/693960de75f58a7b74c835bf?scenarioAccessToken=eyJ..."
  allow="microphone"
  width="100%"
  height="600"
/>
```

Alternatively, you can construct the URL manually by appending the `scenarioAccessToken`
query parameter to the embed URL:

```
https://app.toughtongueai.com/embed/{scenario_id}?scenarioAccessToken={access_token}
```
