# Analytics API

Access aggregated metrics and insights across scenarios and sessions.

## Get Analytics

`POST /v2/analytics`

Retrieve analytics data for specified scenarios and date range.

**Request Body:**

```json
{
  "scenario_ids": ["scenario_1", "scenario_2"],
  "start_date": "2024-01-01",
  "end_date": "2024-01-31",
  "metrics": [
    "completion_rate",
    "avg_score",
    "user_count",
    "total_sessions"
  ]
}
```

**Response:**

```json
{
  "period": {
    "start": "2024-01-01",
    "end": "2024-01-31"
  },
  "metrics": {
    "completion_rate": 0.85,
    "avg_score": 82.5,
    "user_count": 150,
    "total_sessions": 320
  },
  "by_scenario": [...]
}
```

## Available Metrics

- `completion_rate` - Percentage of completed sessions
- `avg_score` - Average analysis score
- `user_count` - Unique users
- `total_sessions` - Total session count
- `avg_duration` - Average session duration
- `engagement_rate` - User engagement metrics
