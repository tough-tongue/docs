# Sessions API

Track and analyze conversation sessions.

## List Sessions

`GET /sessions`

Retrieve sessions with optional filtering.

**Query Parameters:**

- `scenario_id` - Filter by scenario
- `user_email` - Filter by user
- `start_date` - Filter by start date
- `end_date` - Filter by end date

```javascript
const params = new URLSearchParams({
  scenario_id: 'scenario_123',
  start_date: '2024-01-01'
});

const response = await fetch(
  `https://api.toughtongueai.com/api/public/sessions?${params}`,
  {
    headers: { 'Authorization': 'Bearer YOUR_API_TOKEN' }
  }
);

const sessions = await response.json();
```

## Get Session

`GET /sessions/{session_id}`

Retrieve detailed information about a session.

**Response:**

```json
{
  "id": "session_abc123",
  "scenario_id": "scenario_xyz",
  "user_email": "user@example.com",
  "duration": 420,
  "transcript": "...",
  "created_at": "2024-01-15T10:00:00Z"
}
```

## Analyze Session

`POST /sessions/analyze`

Request AI-powered analysis of a session.

**Request Body:**

```json
{
  "session_id": "session_abc123"
}
```

**Response:**

```json
{
  "session_id": "session_abc123",
  "analysis": {
    "overall_score": 85,
    "strengths": [...],
    "improvements": [...],
    "detailed_feedback": "..."
  }
}
```
