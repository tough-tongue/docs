# Interview Preparation App

Learn how to build a complete interview preparation app with ToughTongue AI.

## Overview

This example shows how to create an interview practice platform where users can:
- Practice technical and behavioral interviews
- Receive AI-powered feedback
- Track their progress over time
- Review past sessions

## Architecture

```
Frontend (React/Next.js)
  ↓
Backend API (Your server)
  ↓
ToughTongue AI API
```

## Step 1: Create Interview Scenarios

Create different interview types:

```javascript
const scenarios = [
  {
    name: 'Frontend Developer - Technical',
    ai_instructions: `You are a senior engineering manager conducting a frontend interview. 
    Ask about React, TypeScript, performance optimization, and architecture decisions.
    Be thorough but fair, and ask follow-up questions based on responses.`
  },
  {
    name: 'Behavioral Interview',
    ai_instructions: `You are an HR manager conducting a behavioral interview.
    Ask about past experiences, teamwork, conflict resolution, and leadership.
    Use the STAR method to evaluate responses.`
  }
];

for (const scenario of scenarios) {
  await fetch('/api/scenarios', {
    method: 'POST',
    body: JSON.stringify(scenario)
  });
}
```

## Step 2: Build the Practice Interface

```jsx
export default function InterviewPractice() {
  const [selectedType, setSelectedType] = useState('technical');
  const [sessionId, setSessionId] = useState(null);
  
  return (
    <div>
      <h1>Interview Practice</h1>
      
      <select value={selectedType} onChange={e => setSelectedType(e.target.value)}>
        <option value="technical">Technical Interview</option>
        <option value="behavioral">Behavioral Interview</option>
      </select>
      
      <iframe
        src={`https://app.toughtongueai.com/embed/${getScenarioId(selectedType)}`}
        width="100%"
        height="600px"
        allow="microphone"
      />
      
      {sessionId && <ResultsView sessionId={sessionId} />}
    </div>
  );
}
```

## Step 3: Display Feedback

```jsx
function ResultsView({ sessionId }) {
  const [analysis, setAnalysis] = useState(null);
  
  useEffect(() => {
    fetch(`/api/analyze`, {
      method: 'POST',
      body: JSON.stringify({ session_id: sessionId })
    })
      .then(r => r.json())
      .then(setAnalysis);
  }, [sessionId]);
  
  if (!analysis) return <div>Loading...</div>;
  
  return (
    <div className="results">
      <h2>Your Interview Feedback</h2>
      <div className="score">
        <h3>Overall Score: {analysis.overall_score}/100</h3>
      </div>
      
      <div className="strengths">
        <h4>Strengths:</h4>
        <ul>
          {analysis.strengths.map((s, i) => <li key={i}>{s}</li>)}
        </ul>
      </div>
      
      <div className="improvements">
        <h4>Areas for Improvement:</h4>
        <ul>
          {analysis.improvements.map((i, idx) => <li key={idx}>{i}</li>)}
        </ul>
      </div>
    </div>
  );
}
```
