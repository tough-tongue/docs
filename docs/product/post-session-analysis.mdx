Every session generates detailed analysis based on your scenario's criteria.

## What You Get

- **Full transcript** of the conversation
- **Performance scores** against evaluation criteria
- **Specific feedback** on what to improve
- **Tone analysis** — how you came across

## Evaluation Criteria

By default, AI evaluates:
- General professionalism
- Communication skills
- Response quality

**Custom criteria:** Add specific scoring in your scenario prompt.

Example for sales:
```
Evaluate on:
- Opening hook effectiveness
- Objection handling
- Value articulation
- Call-to-action clarity
```

## Accessing Analysis

1. Complete your session
2. View analysis on the results page
3. Or visit [Sessions](https://app.toughtongueai.com/sessions) anytime

## Exporting Data

**For individuals:**
- Download session transcript
- Export analysis as PDF

**For organizations:**
- **Scenarios CSV:** Scenario-level stats (name, dates, session count)
- **Sessions CSV:** Detailed records (user, timestamps, duration, scores)

Perfect for LMS import, custom reports, and ROI measurement.

## Privacy

Delete any session from the [Sessions page](https://app.toughtongueai.com/sessions).
All data—recordings, metadata, analysis—permanently removed from servers.
