## Creating a Meeting Bot

```javascript
const response = await fetch(
  'https://api.toughtongueai.com/api/public/v2/meeting-bots',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      scenario_id: 'your_scenario_id',
      meeting_url: 'https://zoom.us/j/...',
      bot_name: 'Interview Coach',
      join_at: '2024-01-15T10:00:00Z'
    })
  }
);

const bot = await response.json();
```
