How This Template Works

A complete integration example with ToughTongue AI

1

Embed Voice AI Scenarios

Use iframes to embed any scenario from ToughTongue AI. Users practice conversations right in your app. Configure with query parameters for branding.

2

Listen to Lifecycle Events

JavaScript event listeners capture when sessions start, stop, and provide session IDs. Use window.addEventListener('message', handler) to track user activity.

3

Fetch Analysis via API

Flask API routes proxy requests to ToughTongue AI, keeping your API key secure on the server. Get transcripts, coaching insights, and performance metrics.

4

Display Results & Track Progress

Show users their performance, areas for improvement, and progress over time. Store data in your preferred database.
